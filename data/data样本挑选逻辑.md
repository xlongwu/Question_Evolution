## 样本子集挑选
索引分别是 `859`、`18318`、`6406`、`16877`。

本次筛选严格按照《项目设计文档》中“5.1 第一层：原始数据准入筛选”的标准执行，重点遵循以下规则：

## 优先纳入的样本类型

1. 需要多步推理而非单点回忆；
2. 需要区分“线索 / 可疑 / 待核查 / 可写结论”等不同证据层级；
3. 需要处理监控盲区、时间差、跨摄像头轨迹拼接等问题；
4. 需要比较两个都像关键点的缺口或两种容易混淆的解释路径；
5. 需要在题面事实范围内完成排除性推理；
6. 不依赖题外知识，也不是纯流程背诵题。

## 优先排除的样本类型

1. 一眼就能定性或定罪的题；
2. 纯概念定义题；
3. 单纯背流程即可高分的题；
4. 主要考记忆，不主要考推理或判断的题；
5. 强依赖题外专业知识才能回答的题；
6. 题面信息严重不足，天然容易退化成“无法判断”的题。

## 最终筛中的 4 条

### 1. `859`

题目：
如何通过视频画面判断老人进入监控盲区后是“正常穿过”还是“在盲区内发生意外”？对比这两类情况在时间规律上有何不同？

挑选理由：

1. 这条明显符合“需要处理监控盲区、时间差”的准入条件。
2. 它不是看一眼就能得结论的题，必须结合进入盲区前的状态、正常通行时间、出口重现时间以及周边关联反应来判断。
3. 它要求比较“正常穿过”和“盲区内发生意外”两种解释路径，属于标准的比较分析和边界判断题。
4. 这条不属于纯概念题，也不是单纯背流程即可作答，核心依赖的是时空连续性和异常时间规律推断。
5. 后续很适合向“个体基线”“异常阈值”“哪些前兆最关键”“哪一层判断还不能成立”等方向做 question evolution。

### 2. `18318`

题目：
在处理犬只伤人致死案件中，如何通过视频回溯分析判断该行为属于意外犬只脱缰还是人为故意放任？

挑选理由：

1. 这条不是直接问“有没有伤人结果”，而是要求区分“意外失控”和“人为放任”两种完全不同的解释层级。
2. 要回答好这题，必须综合事前控制行为、脱缰瞬间、脱缰后反应、环境风险和既往先兆，属于明显的多步推理题。
3. 它天然涉及“哪些动作只能说明风险，哪些动作已经能支持主观放任判断”的证据层级问题，符合项目需要的边界推理特征。
4. 该题不是纯记忆题，也不是一眼可判题，因为“有伤人结果”并不能直接推出“故意放任”。
5. 后续可进化成“哪一个子判断还不能成立”“最少还缺哪类动作证据”“哪个现象更能支持放任而不是过失”的高区分度题。

### 3. `6406`

题目：
在办理涉毒案件时，嫌疑车辆进入监控盲区（如偏僻滩地、林间小路）后长时间停留再返回主路。在进行视频研判时，如何通过分析车辆进出监控点的时间差来判断其行为是否存在交易或藏毒嫌疑？

挑选理由：

1. 这条严格符合“监控盲区 + 时间差 + 排除性推理”的优先纳入条件。
2. 它要求建立“正常通行时间基线”和“异常停留时长”之间的比较关系，不是单纯看到车辆停留就能下结论。
3. 该题同时要求排除合法目的地、路径中转、正常临停等其他解释，属于明显的证据层级和排除性推理题。
4. 它不是显然定性题，合理结论只能是“存在交易或藏毒嫌疑，需要补强印证”，这很适合作为后续进化样本。
5. 后续可以继续进化为“哪一个补充事实最关键”“哪一个时间差异常更具有决定性”“哪些现象还不足以写入结论”的题型。

### 4. `16877`

题目：
在研判涉赌案件视频时，如果画面无法直接看清嫌疑人手中拿的是不是现金或筹码，办案民警应重点观察哪些细节动作来辅助判断其行为性质？

挑选理由：

1. 这条不是要求直接看清“物是什么”，而是在题面明确给出“看不清现金还是筹码”的前提下，要求通过连续动作链辅助判断行为性质。
2. 它符合“在题面事实范围内完成排除性推理”的准入条件，因为只能依赖动作规律、角色分工、后续反应和环境伴随物来判断，而不能依赖题外信息。
3. 这条本质上是在区分“看起来像普通递物”与“已经呈现下注、结算、分配逻辑”的差别，属于明显的证据层级问题。
4. 它不是纯定义题，也不是单纯背法规或流程即可回答；如果只会泛泛说“观察手部动作”，通常不足以高质量作答。
5. 后续很适合进化成“哪些动作还只是可疑线索，哪些动作更接近赌博闭环”“哪一类伴随现象最关键”的题。

## 为什么排除了其他看起来也不错的题

本轮有意识地排除了以下几类候选：

### 1. 容易一眼收敛的题

例如之前考虑过的“取保候审人员是否为实际驾驶员”一题，虽然可以做成排他性认定题，但它过于容易被理解成“看谁坐主驾驶位”，存在“一眼就可能得出结论”的风险。因此按照这轮更严格的筛选标准，已不再保留。

### 2. 纯概念或定义色彩偏强的题

例如“某类人员的定义是什么”“某方法的概念是什么”这类题，即使可以扩展，也不符合当前 question evolution 的主攻方向。

### 3. 单纯偏程序或背诵的题

如果一道题主要靠列流程、列注意事项就能高分，但缺少明显的边界判断、证据层级或排除性推理空间，本轮也不优先纳入。

### 4. 题面过于依赖题外补充的题

如果某题要高质量回答必须补充大量题外专业背景，容易在后续进化时退化为“靠外部知识填坑”，这类题也不优先纳入。
